OPENQP

Open Quantum Platform (OpenQP) is a quantum chemical platform featuring cutting-edge capabilities like Mixed-Reference Spin-Flip (MRSF)-TDDFT, with an emphasis on an open-source ecosystem.

For more information, check out the user guide.

This is the documentation for the main branch of OpenQP.
